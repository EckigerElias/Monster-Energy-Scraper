# Monster-Energy-Scraper

Ein Python-Skript zum Scrapen und Verwalten von Monster-Energy Angeboten über kaufda.de.

## Projektstruktur
- `monster_scraper.py`: Hauptskript
- `previous_data.json`: Beispiel-Datendatei

## Nutzung

1. Klone das Repository
2. Wechsle ins Projektverzeichnis
3. Installiere benötigte Abhängigkeiten (falls vorhanden):
   ```bash
   pip install -r requirements.txt
   ```
4. Passe den Code insofern an, ob du Mails oder Discord Nachrichten versenden willst
5. Starte das Skript:
   ```bash
   python monster_scraper.py
   ```
